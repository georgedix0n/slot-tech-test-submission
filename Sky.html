Class: Sky

Sky(cloudCount, cloudTextureNames, cloudAnimationDuration, skyColour, canvasWidth, canvasHeight)

Sky component class that creates a sky including a given number of clouds and animates clouds in the scene.

Constructor

new Sky(cloudCount, cloudTextureNames, cloudAnimationDuration, skyColour, canvasWidth, canvasHeight)

Creates an instance of Sky and initializes it.

Parameters:
Name Type Description
cloudCount number

The number of clouds to generate.
cloudTextureNames Array.<string>

Array of cloud texture names to randomly assign to clouds.
cloudAnimationDuration number

Duration (in seconds) for clouds to traverse the horizontal width of the screen.
skyColour string

The color of the sky, specified as a hexadecimal string.
canvasWidth number

The width of the canvas where the sky will be rendered.
canvasHeight number

The height of the canvas where the sky will be rendered.
